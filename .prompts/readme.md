# AI Prompt Library

This folder contains prompt templates partly used for building this project. 
When using these prompts in Cursor / VS Code Copilot, use the `@` symbol to attach the necessary context.

## Available Prompts

### 1. long_chat_prompt.md (`@long_chat_prompt.md`)
- **When to use:** when needed to move from a chat with very long context window for better organization, accuracy and modular work  
- **Context to attach:** `@docs/______.md` and the relevant `@___/` folder.
